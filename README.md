# 🎬 Movie App

A movie discovery app built with React and TMDB API.

## Features

- Browse popular movies
- Search movies in real-time with debounce
- Filter by genre
- Movie details page
- Save movies to Watchlist

## Tech Stack

- React
- React Router
- Axios
- TMDB API

## Getting Started

1. Clone the repo
2. Create `.env` file with your TMDB token:
   VITE_TMDB_TOKEN=your_token_here
3. Run `npm install`
4. Run `npm run dev`# 🎬 Movie App
